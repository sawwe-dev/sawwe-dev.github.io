What is Python?

Python is a popular general-purpose programming language that can be used for a wide variety of applications. It includes high-level data structures, dynamic typing, dynamic binding, and many more features that make it as useful for complex application development as it is for scripting or "glue code" that connects components together. It can also be extended to make system calls to almost all operating systems and to run code written in C or C++. Due to its ubiquity and ability to run on nearly every system architecture, Python is a universal language found in a variety of different applications.

About Python

Python was conceived in the late 1980s by Guido van Rossum at Centrum Wiskunde & Informatica (CWI) in the Netherlands as a successor to the ABC programming language, which was inspired by SETL, capable of exception handling and interfacing with the Amoeba operating system.

Variables are like containers, used to store values. To create a variable, write the name of the variable and asign it a value:

>>> variable_name = value

>>> message = "Hello" >>> Number = 9
Data Types

Python has the following data types by default:

  • Str(string) Used to store text:

    >>> my_str = "Hello World"

  • Int(Integer) Used to store numbers without decimal points:

    >>> my_int = 5

  • Float(Floating-point) Used to store numbers with decimal points:

    >>> my_float = 3.14

  • Bool(Boolean) Used to store binary data, that can only be one of two possible values (True or False):

    >>> my_bool = True

  • List(string) Used to store multiple items in a single variable:

    >>> my_list = [1, "dog", 20.8, True]"

We use type() function to return the data type of a variable:

>>> print(type(my_str)) <class 'str'>
Operators

Comparision Operators

In Python, we use Comparison operators to compare a value with another value, the basic comparison operators are:

  • Less than (<)

  • Less than or equal (< =)

  • Greater than (>)

  • Greater than or equal to (> =)

  • Equal to (= =)

  • Not equal to (! =)

Example, comparing two integers:

>>> int1 = 5 >>> int2 = 10 >>> print(int1 < int2) >>> print(int1 == int2) True #Is true that 5 is less than 6 False #Is not true that 5 = 6

Logical Operators

We use logical operators to check multiple conditions at the same time, Python logical operators are:

  • and Used to check if two conditions are true at the same time

    >>> 6 > 2 and 7 == 7 True

  • or Used to check if one on check if either or both conditions are true

    >>> 2 > 1 or 9 == 3 True

  • not Reverses the result of a condition

    >>> not False True
Statements and loops

In Python we use statements and loops to control the logic and flow of the code, basic statements and loops are:

  • If Else statement: We use this to execute a piece of code if a condition is true:

    >>> int1 = 3 >>> int2 = 9 >>> if int1 < int2 : print("int1 is smaller than int2") int1 is smaller than int2

  • For loop: We use the for loop execute a piece of code multiple times, if a condition is true:

    >>> my_list = ["dog", "cat", 5, 23, 90.7] >>> for element in my_list: print(element) #for each element in my_list variable, print the element. This would repeat for every element in the list dog cat 5 23 90.7

  • While loop: We use this loop to repeat a piece of code as long as a condition is true

    >>> my_bool = True >>> While my_bool: print("The condition is true") The condition is true The condition is true The condition is true ... #This code will run forever because my_bool (the condition) will allways be true
